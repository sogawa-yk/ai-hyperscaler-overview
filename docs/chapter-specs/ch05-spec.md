# 第5章 データ側AI（DB組み込み）― DB内ベクトル・Select AI・データ層認可 ― 仕様書

## 章の概要

**章の目的**: 本書の重心の一つで、読者のギャップ埋めを兼ねる。「データの中で動くAI」を、DB内ベクトル・SQLからのLLM／NL→SQL・データ層のきめ細かい認可・ハイブリッド検索・レイクハウス横断ポリシーの軸で切り、4社をプロットする。Oracleの主戦場を正直な出典主義で扱う。

**目標ページ数**: 30p

**所属する部**: 第III部 AIの中核

## 前提知識

この章を読むために必要な前提:
- 序章の方法論
- 第1章（委譲・ID伝播）― データ層認可の前提
- 第3章（レイクハウス接続）、第4章（マネージドRAGとの対比）
