# 第6章 観測性／ガバナンス ― クラウドネイティブo11y・LLM特化o11y・AIゲートウェイ ― 仕様書

## 章の概要

**章の目的**: AIワークロードを「見張る・統治する」領域を、クラウドネイティブo11y・LLM/エージェント特化o11y・AIゲートウェイ・行動監査の軸で切り、4社をプロットする。クラウド非依存の共通層（OTel GenAI semconv / Langfuse）も軸に位置づける。

**目標ページ数**: 22p

**所属する部**: 第IV部 運用・統合・実践
